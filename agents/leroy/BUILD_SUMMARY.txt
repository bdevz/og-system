================================================================================
SKILLS DELIVERED (6 TOTAL)
================================================================================

1. HEALTH CALCULATOR
   - Scores 7 factors: account age, connections, daily apps, restrictions, 
     activity diversity, session patterns, acceptance rate
   - Outputs: 0-100 score + GREEN/YELLOW/ORANGE/RED state
   - Batch mode for portfolio snapshots
   - Fully configurable weights in JSON
   
2. ACTIVITY SIMULATOR
   - Generates randomized daily schedules per profile per tier
   - Respects health state constraints
   - Tier-specific activity mixes (A/B/C/D)
   - Randomized timing, gaps, activity order
   - Weekend reduction (50% probability)

3. INBOUND CLASSIFIER
   - 5-class classification: A (HOT), B (WARM), C (NETWORKING), D (SPAM), E (SUSPICIOUS)
   - SLA: Class A in 15 min, Class B in 2 hours, Class E immediate
   - Keyword-based with pattern matching
   - Extracts opportunity summary and sender context
   - Confidence scoring (0.5-1.0)

4. APPLICATION EXECUTOR
   - Pre-flight checks: health, daily limit, time gap, proxy, Z approval
   - 6-stage workflow: navigate, read, parse, fill, submit, confirm
   - Timing simulation (45-130 seconds realistic duration)
   - Screenshot capture for audit trail
   - Error handling with escalation

5. CONNECTION MANAGER
   - Tier-specific daily limits (A=20, B=15, C=8, D=0)
   - Target distribution strategies
   - Note strategy: 70% no note, 20% generic, 10% personalized
   - Batch execution with statistics
   - Acceptance rate tracking

6. PROFILE LIFECYCLE
   - Warming protocol: 4 phases over 180 days
     Week 1-2: 50 connections
     Week 3-4: 100 connections
     Month 2-3: 150 connections
     Month 4-6: Ready for Tier B
   - Repositioning engine: 5-7 day spread
     Day 1: Headline, Day 2: Summary, Day 3: Skills
     Day 4: Featured, Day 5: Experience
   - Safety constraints: 14 day min age, 30 day min between
