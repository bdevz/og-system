# Do-Not-Submit Master List

This file is the consolidated DNS list across all consultants. Updated by Z when DNS entries change. Used by conflict-checker as a secondary check alongside per-consultant DNS lists.

## Format

| Consultant ID | Company | Vendor (if specific) | Reason | Date Added | Added By |
|---|---|---|---|---|---|
| | | | | | |

## Rules
- Entries can be company-wide (applies to all vendors) or vendor-specific.
- Removing an entry requires human approval and a logged reason.
- This list is checked on every submission request.
